Политика конфиденциальности

City Planner – Card Deck Helper
Дата вступления в силу: 01.12.2025

Это неофициальное приложение-помощник для настольной игры и не связано с издателями оригинального продукта.

⸻

1. Сбор данных

1.1. Личные данные

Приложение не собирает личные данные — имя, email, телефон, фото, контакты, точное местоположение и т. д.

1.2. Данные, собираемые автоматически

Для улучшения качества работы используются сервисы Firebase:

Firebase Analytics
Собирает обезличенные данные:
	•	события и действия в приложении;
	•	тип устройства, модель, версия ОС;
	•	регион (без точной геолокации);
	•	технические параметры для анализа производительности.

Firebase Crashlytics
Собирает данные о сбоях:
	•	стек ошибок;
	•	состояние приложения во время сбоя;
	•	модель устройства и версия ОС;
	•	версия приложения.

Crashlytics не собирает информацию, позволяющую идентифицировать личность.

⸻

2. Использование данных

Данные применяются для:
	•	анализа поведения пользователей (только агрегировано);
	•	улучшения стабильности и производительности приложения;
	•	поиска и исправления ошибок.

Персонализированная реклама отсутствует.

⸻

3. Передача данных

Данные передаются только сервисам Firebase (Google).
Мы не продаём и не передаём данные третьим лицам.

⸻

4. Разрешения

Приложение не запрашивает доступ к:
	•	камере;
	•	микрофону;
	•	контактам;
	•	файлам;
	•	точной геолокации.

⸻

5. Хранение данных

Обезличенные данные хранятся на серверах Google в соответствии с их политиками безопасности.

⸻

6. Безопасность

Используются встроенные инструменты безопасности Firebase и стандартные технические меры.

⸻

7. Дети

Приложение может использоваться детьми, так как личные данные не собираются.
Firebase может обрабатывать технические данные устройства.

## Privacy Policy

City Planner – Card Deck Helper
Effective date: December 1, 2025

This is an unofficial companion app and is not affiliated with the publishers of the original board game.

⸻

1. Data Collection

1.1. Personal Data

The App does not collect personally identifiable information such as name, email, phone number, photos, contacts, or precise location.

1.2. Automatically Collected Data

The App uses Firebase services:

Firebase Analytics
Collects anonymized aggregated data:
	•	in-app events and interactions;
	•	device type, model, OS version;
	•	region-level location (no precise GPS);
	•	performance and usage metrics.

Firebase Crashlytics
Collects crash diagnostic information:
	•	crash logs and stack traces;
	•	app state during the crash;
	•	device model and OS version;
	•	app version.

Crashlytics does not collect personal identifiers.

⸻

2. Use of Data

Data is used to:
	•	analyze app usage in aggregate form;
	•	improve performance and stability;
	•	identify and fix crashes.

No personalized advertising is used.

⸻

3. Sharing of Data

Data is shared only with Firebase (Google) services.
We do not sell or share data with third parties.

⸻

4. Permissions

The App does not access:
	•	camera;
	•	microphone;
	•	contacts;
	•	files;
	•	precise location.

⸻

5. Data Storage

Firebase stores anonymized data on Google servers under their security policies.

⸻

6. Security

We rely on Firebase security features and standard protection mechanisms.

⸻

7. Children

The App can be used by children, as no personal data is collected.
Firebase may process basic device information.
